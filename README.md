# web-dev-exercises

This is a collection of all my small projects and exercises that I'm working on as I'm mastering web development. It’s mostly a place for me to keep track of my progress and have all my practice files in one spot rather than scattered across my computer.

Right now, the focus is on getting the fundamentals down using **HTML, CSS, and vanilla JavaScript**. Most of the folders in here are just quick UI tests, CSS layout practices, or small JS logic puzzles. They aren't meant to be polished "apps," just the raw work I’m doing to build muscle memory.

Since this repo is strictly for practice, I’m keeping my more "complete" work elsewhere. My best project so far—the calculator—is in its own repository because it’s a bit more involved than the quick exercises you’ll find in here. 

If you're a friend checking this out, feel free to look through the code. It’s definitely a work in progress, and I’ll be adding more as I keep learning and breaking things.
